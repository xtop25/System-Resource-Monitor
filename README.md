System-Resource-Monitor
- A lightweight C++ utility to monitor system RAM usage in real-time. Designed to demonstrate efficient memory handling and system interaction using WinAPI.


Features
- Real-time monitoring: Tracks physical RAM usage.

- Lightweight: Minimal CPU and memory footprint.

- Clean Output: Real-time console update without clutter.


Requirements
OS: Windows 10/11.


Compiler: Any C++ compiler (Visual Studio recommended).


Usage
- Compile the code using Visual Studio or your preferred C++ compiler.

- Run the resulting .exe file.

- The console will display the current RAM usage in percentage and megabytes.
